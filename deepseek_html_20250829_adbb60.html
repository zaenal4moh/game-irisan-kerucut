Game Identifikasi Irisan Kerucut

Skor: 0

Identifikasi jenis irisan kerucut dari persamaan berikut:

(x - 2)² + (y + 3)² = 16

Ciri-ciri Persamaan Irisan Kerucut:

  • Lingkaran: Bentuk umum x² + y² = r² atau (x-h)² + (y-k)² = r²
  • Elips: Bentuk umum (x-h)²/a² + (y-k)²/b² = 1
  • Parabola: Bentuk umum y² = 4px atau x² = 4py
  • Hiperbola: Bentuk umum (x-h)²/a² - (y-k)²/b² = 1 atau (y-k)²/a² - (x-h)²/b² = 1
